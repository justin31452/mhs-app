MHS-APP
===

## Usage

> build at Mac OS Catalina 10.15.6, Node.js v12.18.2, expo-cli 3.22.1

1. Install Nodejs.
2. Install Expo-cli.
3. Clone repository.
4. Cd to MHS-APP.
5. Run $npm install.
6. Run $expo start.
